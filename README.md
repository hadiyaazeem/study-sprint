#  Study Sprint

A modern and responsive focus timer application built with React and TypeScript to help students and professionals stay productive using timed study sessions.

---

## Features

-  Pomodoro-style focus timer
-  Start, Pause & Resume sessions
-  Reset timer
-  Progress bar
-  Light & Dark mode
-  Session statistics
-  Session history log
-  Responsive design
-  Clean modern UI
-  Session saving support

---

##  Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- CSS Variables
- Local Storage

##  Installation

Clone the repository

```bash
git clone https://github.com/hadiyaazeem/study-sprint.git
```

Move into the project

```bash
cd study-sprint
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

##  Purpose

Study Sprint was developed to provide a distraction-free productivity experience through structured study sessions. The application helps users maintain focus, track completed sessions, and build consistent study habits.
